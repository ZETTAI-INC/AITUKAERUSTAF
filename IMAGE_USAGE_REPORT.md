## 画像ディレクトリ構造

```
img/
├── common/                    # 共通アセット
│   ├── favicon.ico           # ファビコン
│   ├── logo-header.svg       # ヘッダーロゴ
│   ├── logo-footer.svg       # フッターロゴ
│   ├── button-icon-siryo.svg # 資料請求アイコン
│   ├── button-icon-contact.svg # お問い合わせアイコン
│   ├── footer_fb.svg         # Facebookアイコン
│   ├── footer_x.svg          # Xアイコン
│   ├── p-mark.png            # プライバシーマーク
│   └── [その他アイコン]
│
├── index/                     # ホームページ用
│   ├── top-img.svg           # メインビジュアル
│   ├── about-icon01〜03.svg  # 特徴アイコン
│   ├── about-chart.png       # チャート図
│   └── solution-img01〜04.png # ソリューション画像
│
├── pages/                     # 各ページ用
│   ├── icon-service01〜06.svg     # サービスアイコン（小）
│   ├── icon-service01〜06-s.svg   # サービスアイコン（大）
│   ├── service-img01〜06.png      # サービス説明画像
│   ├── merit-img01〜04.png        # メリット説明画像
│   ├── price-company-icon-*.svg   # 価格プランアイコン
│   └── ai_tools.jpg               # AIツール一覧
│
└── uploads/                   # アップロード画像
    ├── [会計システムロゴ 39種類]
    ├── [導入事例画像 10社以上]
    └── 2025/10/Group-354.png  # アニメーション背景
```
